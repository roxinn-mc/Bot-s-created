# Criar bot do Minecraft
Arquivo para criar bot de Minecraft (disponível apenas no Windows 10 e 11; não disponível para outros sistemas operacionais)

# Tutorial:
1. na area de trabalho do seu computador, clique no botão de direito do mouse, clique em novo e depois em pasta, agora coloque o nome do seu robô (recomendado)
2. entre na pasta, e baixe o arquivo index.js e configure o aquivo (o arquivo é o "cérebro" do seu robô)
3. depois de comfigurá-lo, asesse: https://ollama.com/ e baixe a ollama ( ollama é responsável por colocar a IA/inteligêcia artificial; ela é pesada, então vai demorar o aixamento dela e deixe mais de 20 GB liberado!)
4. agora entre em: https://nodejs.org/ e baixe e instale o node.js (ele é responsavel por criar o robô; ele nn é pesado como a ollama)
